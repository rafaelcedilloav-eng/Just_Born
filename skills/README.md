# skills/ — habilidades regalables · giftable skills

*(ES)* **En curaduría.** Aquí irán habilidades (skills) que construimos en nuestro trabajo real y que pueden servirle a cualquiera — reestructuradas y limpiadas como regalo. Cada una pasará tres filtros antes de entrar: ¿referencia infraestructura o proyectos privados? (fuera o se genericiza) · ¿enseña un patrón reusable? (dentro) · ¿está a la altura de un regalo? (se pule primero).

*(EN)* **Under curation.** This folder will hold skills we built in real work that can serve anyone — restructured and cleaned as a gift. Each one passes three filters before entering: does it reference private infrastructure or projects? (out, or genericized) · does it teach a reusable pattern? (in) · is it up to gift standard? (polished first).

*Primeras candidatas / First candidates: patrones de debugging de training de modelos, auditoría QA sin suavizar hallazgos, filosofía de diseño frontend.*
